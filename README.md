# Тайное голосование на Cloudflare Pages

Простая система тайного голосования для небольших выборов.

## Что умеет

- админка с паролем из env/secrets;
- создание голосований;
- генерация одноразовых кодов доступа;
- старт, остановка и закрытие голосования;
- голосование по коду доступа;
- отдельный код проверки бюллетеня;
- итоговый протокол и JSON/CSV-экспорт;
- D1 как рабочее хранилище, без связи access code и бюллетеня.

## Локальный запуск

```bash
npm install
npm run build
npm test
npm run dev
```

## Деплой на Cloudflare Pages

Для Pages проекта используйте:

```bash
npm run deploy:pages
```

Не используйте `wrangler deploy` для этого репозитория: это команда для Worker-проекта, а здесь нужен Pages-деплой из папки `public`.

## Переменные окружения

- `ADMIN_PASSWORD` - пароль для админки.
- `APP_SECRET` - секрет для шифрования временных токенов.
- `CODE_SALT` - соль для хэшей одноразовых кодов.
- `HOME_PASSWORD` - пароль для входа на главную страницу.

## D1

Схема лежит в [`schema.sql`](./schema.sql). Для локальной разработки поднимите D1 binding в `wrangler.jsonc` и выполните схему через Wrangler.
